THERE ARE THREE TYPES OF CSS

INLINE CSS:- An inline CSS is used to apply a unique style to a single HTML element. An inline CSS uses the style attribute of an HTML element

INTERNAL CSS:- An internal CSS is used to define a style for a single HTML page. An internal CSS is defined in the section of an HTML page, within a style element.

EXTERNAL CSS:- External CSS is a form of CSS which is used to add styling to multiple HTML pages at a time. It helps to design the layout of many HTML web pages simultaneously. The external CSS is always saved with the . css extension, and through this file, we can change the complete style of our HTML web page
